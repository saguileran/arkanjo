\doxysubsection{Detailed Description}
Main function extraction processor. 

Analyzes source code directories (C/\+C++/\+Java) and extracts each function into individual files within a temporary repository. Handles file organization and delegates language-\/specific parsing to appropriate handlers.

\begin{DoxyNote}{Note}
Current limitations\+:
\begin{DoxyItemize}
\item May not work correctly with malformed bracket sequences
\item May fail on brackets within comments
\item Example behavior\+: For file example.\+c with functions a and b, creates example/a.\+c and example/b.\+c in tmp/ directory 
\end{DoxyItemize}
\end{DoxyNote}
