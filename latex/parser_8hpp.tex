Code duplication results parser. 

Handles parsing and transformation of output from third-\/party code duplication detection tools into a more usable format.

The parser is responsible to read the output to the code duplication detection tool that exists in the third\+\_\+party folder and parse/transform it in a way that is easier to work with later.
